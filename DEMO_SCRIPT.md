# Waykin MPOC Demo Script

The complete proof loop must be demonstrable in five minutes without location permission, external credentials, or physical movement.

## Scenario 1 — Calm Day Walk

1. Launch Demo Mode.
2. Create or select a companion.
3. Select `WALK` and `COMPANION_WALK`.
4. Run `CALM_DAY_WALK`.
5. Observe follow/lead behavior, milestone reaction, and bond gain.
6. Complete the session.
7. Confirm summary and memory creation.

## Scenario 2 — Night Orc Pursuit

1. Select `RUN` and `ORC_PURSUIT`.
2. Run `NIGHT_ORC_PURSUIT`.
3. Observe pursuers gain while pace falls.
4. Pause and confirm unfair capture does not occur.
5. Resume, accelerate, and observe threat reduction.
6. Complete with a deterministic outcome.

## Scenario 3 — Future Self Interval

1. Select `RUN` and `FUTURE_SELF`.
2. Run `FUTURE_SELF_INTERVAL`.
3. Observe bounded adaptive lead and target pace.
4. Enter a catch window.
5. Complete the session and persist prior pace.

## Continuity Proof

1. Terminate and relaunch the app or recreate the persistence container.
2. Confirm companion, bond, sessions, and memories remain available.
3. Change the controllable clock from day to night.
4. Confirm recommendation order and explanation change.

## Required Evidence

- Build command and result
- Test command and result
- Demo scenario outcomes
- Persistence evidence
- Permission-denial evidence
- Any unperformed step labeled `NOT_COMPUTABLE`
